## Overview

AetherOS is a modern Linux-based operating system focused on productivity, intelligent computing, and a futuristic desktop experience.

Rather than reinventing Linux itself, AetherOS builds upon a stable Linux foundation while introducing an entirely redesigned user experience powered by modern UI design and AI.

## Installation

Install Virtual Box
create new machine called "AetherOS"
get the .iso file in it
double click setup_vm.ps1
double click AetherOS.bat

Enjoy.
